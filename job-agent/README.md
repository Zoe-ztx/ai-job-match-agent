# 职引 · AI求职智能匹配智能体

## 📋 项目概述

**职引**是一款基于AI大模型的求职智能匹配系统，旨在帮助应届毕业生精准匹配就业岗位、优化简历内容、分析同背景求职者去向，提升简历初筛命中率。

### 核心功能

1. **岗位智能匹配** - 基于用户背景智能推荐最合适的岗位，分析匹配度与补强建议
2. **简历优化诊断** - AI深度分析简历不足，给出针对性改进建议和改写示例
3. **同背景去向分析** - 了解同背景求职者的就业去向、城市分布和薪资水平

### 技术栈

**前端：**
- HTML5 + CSS3 (紫色/金色主题)
- Vanilla JavaScript (无框架依赖)
- 响应式设计，支持移动端

**后端：**
- Node.js + Express
- 文件上传：multer + pdf-parse + mammoth
- AI调用：HTTPS + 流式响应
- 搜索增强：SerpAPI (Google搜索)

**AI模型：**
- 腾讯混元 (hunyuan-turbos-latest) - 推荐
- 智谱AI (glm-4-flash) - 备选

## 🚀 快速开始

### 环境要求

- Node.js 18+ 
- npm 或 yarn
- AI大模型API Key (腾讯混元或智谱AI)

### 安装步骤

```bash
# 1. 克隆项目
cd job-agent

# 2. 安装依赖
npm install

# 3. 配置环境变量
# 方式一：创建.env文件
echo "LLM_PROVIDER=hunyuan" > .env
echo "HUNYUAN_API_KEY=your_api_key_here" >> .env

# 方式二：直接设置环境变量
export LLM_PROVIDER=hunyuan
export HUNYUAN_API_KEY=your_api_key_here

# 4. 启动服务器
npm start
# 或
node server.js
```

### 访问应用

打开浏览器访问：`http://localhost:3000`

## 📚 功能详解

### 1. 岗位智能匹配

**功能描述：**
- 上传简历或手动输入背景信息
- AI自动提取关键信息（学历、技能、项目、实习）
- 基于用户背景智能推荐5-8个最匹配岗位
- 每个岗位包含：公司、城市、薪资、匹配度、技能匹配分析、投递链接

**技术实现：**
- 简历解析：支持PDF/DOC/DOCX/TXT格式
- AI匹配：调用大模型生成岗位推荐
- 搜索增强：通过SerpAPI搜索真实岗位信息
- 链接修正：自动校验和修正投递链接

**预估时间：** 约60秒

---

### 2. 简历优化诊断

**功能描述：**
- 上传简历进行AI诊断
- 生成简历评分（0-100分）
- 识别简历中的问题（错误/警告）
- 提供具体改进建议和改写示例
- 对比改写前后的差异

**技术实现：**
- 多维度评分：内容完整性、技能匹配度、经历描述、排版规范性
- 问题识别：拼写错误、排版问题、内容缺失、表达不当
- 改写示例：AI生成优化后的表达方式

**预估时间：** 约60秒

---

### 3. 同背景去向分析

**功能描述：**
- 输入个人背景信息（学历、专业、技能）
- AI分析同背景求职者的就业去向
- 展示热门就业城市及人数分布
- 分析薪资水平区间
- 提供同背景成功案例

**技术实现：**
- 数据搜索：通过SerpAPI搜索相关就业报告
- AI分析：调用大模型分析搜索结果
- 结构化输出：城市分布、薪资范围、岗位类型、求职建议

**预估时间：** 约60秒

## 🔧 API接口文档

### 健康检查
```
GET /api/health
响应：{ "status": "ok", "provider": "hunyuan", "apiKeyConfigured": true }
```

### 上传简历
```
POST /api/upload-resume
Content-Type: multipart/form-data
Body: resume=(文件)
响应：{ "text": "简历文本内容", "error": "错误信息" }
```

### 解析简历
```
POST /api/parse-resume
Body: { "resumeText": "简历文本", "sessionId": "session_xxx" }
响应：{ "parsed": { "edu": "", "skills": "", ... }, "error": "..." }
```

### 设置上下文
```
POST /api/set-context
Body: { "sessionId": "xxx", "userInfo": {...}, "func": "match" }
响应：{ "success": true }
```

```
POST /api/chat

```

## 🐛 已知问题

1. **PDF解析可能失真** - 复杂排版PDF解析结果可能不准确，建议同时使用TXT格式
2. **AI生成时间较长** - 大模型生成需要时间，已添加倒计时提示
3. **搜索结果依赖SerpAPI** - 需要网络连接，免费Key有配额限制
